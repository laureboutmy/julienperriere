1Mission

We built a full featured video player designed to enrich the viewing experience of the fans with social interactions, additional contents poping in sync with the episode and a gamification grid to reward users’ actions.

GoT Player is a school projet done with Laure Boutmy, Dorian Camilleri, Thomas Iturralde and Anthony Roux. We imagined a scenario where HBO would come to us with the intention to create a digital event to promote their TV series, create engagement and collect data.

This project was very impressive. I created both the UX and the UI design, and I had a lot of fun all along the process : from defining the top-level framework to sweating the interface details to make sure it didn’t attract the eye too much.

2Countdown

In our scenario, the episode would be available online at the same time it’s broadcasted on tv. so i created a landing page where people can start collecting badges by signing up with their social networks.

Countingthe hours

The landing page features a countdown to the exact time the episodes is viewable on the website, to make sure they don’t miss it.

Game of Thronesbadges set

I wanted three things for the badges : to fit well within the interface, to look a bit like actual badges and to be beautiful - after all, they are rewards. The seven corners? For the seven kingdoms, but you had already guessed it.

3Player

It went through a lot of iterations to create the design of the player. the challenge was to find the perfect balance between an interface that’s easy to use and encourages interactions, and optimal watching conditions.

Navigating throughthe content

Additional contents are symbolized by dots that pop as soon as you’ve unlocked them. Even I wanted the interface to be easy to use and discrete, I also wanted it to look GoT-ish.

Sidebarcards

Information about a character, a place or an event, statistics, GIFs, bonus videos, soundtrack, quizzes... these are the types of contents that I had to find a common template to showcase within the sidebar in a pleasing and inviting way.

Going furtherthe browser

It’s the name I gave to this section that deploys when this user wants to know more about a certain type of content : read a biography, watch a making-of video or share a GIF on Tumblr.
